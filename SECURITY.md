# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TeleClaw, please report it responsibly.

**Do not** open a public GitHub issue.

Instead, please email: security@teleclaw.dev

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.4.x   | ✅ |
| 1.3.x   | ✅ |
| < 1.3   | ❌ |

## Security Features

- On-device processing by default
- Shizuku service invisible to other applications
- No telemetry or data collection
- API keys stored locally with encryption
