# Contributing to TeleClaw

Thank you for your interest in contributing to TeleClaw!

## Current Status

At this time, the core agent modules (conversation engine, memory system, tool orchestration, virtual display) are **proprietary and closed-source**. The code in this repository represents the open framework layer.

## How You Can Help

- 🐛 **Bug Reports** — Open an [Issue](../../issues) with steps to reproduce
- 💡 **Feature Requests** — Open an [Issue](../../issues) with the `feature` label
- 📖 **Documentation** — Typos and clarifications are welcome
- ⭐ **Stars** — Help others discover TeleClaw

## Code Contributions

We are not accepting pull requests that modify core agent functionality at this time. Framework-level improvements (build scripts, documentation, CI) may be considered.

## Code of Conduct

Be respectful. Be constructive. No spam.
